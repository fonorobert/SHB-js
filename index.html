Example of SHB.js. See more at How to Social Buttons in HTML only using Twitter Bootstrap 3 and Font Awesome Icons (No Javascript).

The SHB.js script build on the fly HTML social buttons, which based on Twitter Bootstrap 3 markup ant Font Awesome icons.

Build All Default Buttons

SHB.build('shb');

Build All Default Buttons with custom size

SHB.build({
  elementID: 'shb_otions',
  
  pref: {
  	btnSizeClass: 'btn-sm',
  }
});

Build Some of Default Buttons And One Custom (GitHub)

SHB.build({
  elementID: 'shb_custom',
	buttons: {
  	fbLike: true,
		tweet: true,
  	plusOne: true,
		surf: true,
  
  	customButton: {
  		iconClass: 'fa-github',
  		iconURL: 'https://github.com/dr-dimitru',
  		iconParams: {} 
  	},
	}
});
